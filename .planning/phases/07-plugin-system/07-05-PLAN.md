    1. Export root module using abi_stable macros:
    ```rust
    use abi_stable::prelude::*;
    use abi_stable::{export_root_module, sabi_extern_fn};
    use abi_stable::std_types::{RResult, RString, RVec, ROk, RErr};
    use slicecore_plugin_api::{
        InfillPatternPlugin, InfillPatternPlugin_TO,
        InfillPluginMod, InfillPluginMod_Ref,
        InfillRequest, InfillResult, FfiInfillLine,
    };

    #[export_root_module]
    fn instantiate_root_module() -> InfillPluginMod_Ref {
        InfillPluginMod {
            new: new_plugin,
        }.leak_into_prefix()
    }

    #[sabi_extern_fn]
    fn new_plugin() -> InfillPatternPlugin_TO<'static, RBox<()>> {
        InfillPatternPlugin_TO::from_value(ZigzagInfillPlugin, TD_Opaque)
    }
    ```

    2. Implement the zigzag algorithm:
    ```rust
    #[derive(Debug, Clone)]
    struct ZigzagInfillPlugin;

    impl InfillPatternPlugin for ZigzagInfillPlugin {
        fn name(&self) -> RString {
            "zigzag".into()
        }

        fn description(&self) -> RString {
            "Zigzag infill: continuous diagonal lines that bounce between boundaries".into()
        }

        fn generate(&self, request: &InfillRequest) -> RResult<InfillResult, RString> {
            // Decode boundary into polygon points
            // Compute bounding box from boundary_points
            // Generate zigzag scan lines at density-based spacing
            // Clip against polygon boundary
            // Connect alternating endpoints to form continuous zigzag
            // Return InfillResult with FfiInfillLine segments
        }
    }
    ```

    The zigzag algorithm:
    - Compute bounding box of all boundary points
    - Calculate line spacing = line_width / density
    - Generate vertical scan lines from min_x to max_x at spacing intervals
    - For each scan line, find intersections with polygon edges
    - Sort intersections by Y
    - Create line segments between pairs of intersections
    - Connect adjacent scan line segments at top/bottom alternately to form continuous zigzag path

    Keep the algorithm simple but functional (this is an example, not a production pattern). Use the flattened boundary_points and boundary_lengths to reconstruct polygon edges for intersection testing.

    Implement a simple `point_in_polygon_scanline` helper that finds Y intersections of a vertical line with polygon edges using the boundary_points/boundary_lengths format.

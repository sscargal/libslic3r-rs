1. Add `robust = "1.2"` to slicecore-mesh Cargo.toml dependencies.

2. Create `crates/slicecore-mesh/src/csg/mod.rs` as the module root. Declare submodules: `pub mod error;`, `pub mod report;`, `pub mod attributes;`, `pub mod primitives;`. Re-export key types: `CsgError`, `CsgReport`, `BooleanOp`, `TriangleAttributes`.

3. Create `csg/error.rs` with `CsgError` enum using thiserror. Variants per RESEARCH.md:
   - `RepairFailedA(#[source] MeshError)` — mesh A repair failed
   - `RepairFailedB(#[source] MeshError)` — mesh B repair failed
   - `EmptyResult { operation: String }` — boolean produced no triangles
   - `IntersectionFailed { tri_a: usize, tri_b: usize, reason: String }` — intersection computation failed
   - `ResultConstruction(#[source] MeshError)` — output mesh construction failed
   - `Cancelled` — operation cancelled

4. Create `csg/report.rs` with `CsgReport` struct deriving `Debug, Clone, Default, Serialize, Deserialize`. Fields: `input_a_triangles: usize`, `input_b_triangles: usize`, `output_triangles: usize`, `intersection_curves: usize`, `repairs_performed: usize`, `warnings: Vec<String>`.

5. Create `csg/attributes.rs` with:
   - `TriangleAttributes` struct: `material_id: u32`, `color: Option<[u8; 4]>`. Derive `Clone, Debug, Default, Serialize, Deserialize, PartialEq`.
   - Helper function `propagate_attributes(source_attrs: Option<&[TriangleAttributes]>, tri_idx: usize) -> TriangleAttributes` that returns the attribute for a given source triangle or `Default` if no attributes.

6. Create `BooleanOp` enum in `csg/mod.rs`: `Union, Difference, Intersection, Xor`. Derive `Debug, Clone, Copy, PartialEq, Eq`.

7. Extend `TriangleMesh` in `triangle_mesh.rs`:
   - Add field `attributes: Option<Vec<TriangleAttributes>>` (import from `csg::attributes`).
   - Add `pub fn attributes(&self) -> Option<&[TriangleAttributes]>` getter.
   - Add `pub fn with_attributes(mut self, attrs: Vec<TriangleAttributes>) -> Result<Self, MeshError>` that validates length matches triangle count.
   - Add `pub fn set_attributes(&mut self, attrs: Option<Vec<TriangleAttributes>>)` — for clearing or setting.
   - Initialize `attributes: None` in `TriangleMesh::new()` — backward-compatible.
   - Update `unit_cube()` to set `attributes: None`.

8. In `lib.rs`, add `pub mod csg;` and re-export: `pub use csg::{CsgError, CsgReport, BooleanOp};`.

All public items must have doc comments. Use `#![warn(clippy::all, clippy::pedantic)]` allows at csg module level only if needed for specific pedantic lints (document why).

  <action>
Create `csg/primitives.rs` with these public functions:

- `pub fn box_mesh(width: f64, height: f64, depth: f64) -> TriangleMesh` — axis-aligned box centered at origin. 8 vertices, 12 triangles. Outward-facing normals.
- `pub fn cylinder(radius: f64, height: f64, segments: u32) -> TriangleMesh` — centered at origin, extends from z=0 to z=height. Default segments=32 if 0 passed. Bottom cap, top cap, side quads as triangle pairs. Use `segments.max(3)`.
- `pub fn sphere(radius: f64, segments: u32, rings: u32) -> TriangleMesh` — centered at origin. UV-sphere with poles. Default segments=16, rings=8. Use `segments.max(3)`, `rings.max(2)`.
- `pub fn cone(radius: f64, height: f64, segments: u32) -> TriangleMesh` — apex at z=height, base at z=0. Default segments=32.
- `pub fn torus(major_radius: f64, minor_radius: f64, major_segments: u32, minor_segments: u32) -> TriangleMesh` — centered at origin in XY plane. Default 32 for both.
- `pub fn plane_halfspace(normal: Vec3, distance: f64, extent: f64) -> TriangleMesh` — generates a large thin box (thickness ~0.001) perpendicular to normal at signed distance. Used as CSG operand for plane cuts. `extent` controls the XY size.
- `pub fn wedge(width: f64, height: f64, depth: f64) -> TriangleMesh` — triangular prism (ramp shape). 6 vertices, 8 triangles.

All primitives generate watertight meshes with outward-pointing normals and consistent CCW winding when viewed from outside. All use `TriangleMesh::new()` constructor.

Write tests in `crates/slicecore-mesh/tests/csg_primitives.rs` covering the behaviors listed above.

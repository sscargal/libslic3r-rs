<task type="auto">
  <name>Task 2: Nine mesh primitive generators</name>
  <files>crates/slicecore-mesh/src/csg/primitives.rs</files>
  <action>
    Create `crates/slicecore-mesh/src/csg/primitives.rs` with 9 primitive mesh generators. Each returns `TriangleMesh` (infallible for valid params). All generate meshes centered at origin. All must produce watertight (manifold) meshes with correct outward-facing normals.

    Default tessellation: `segments: u32 = 32` for curved surfaces.

    1. `primitive_box(width: f64, height: f64, depth: f64) -> TriangleMesh`
       - Axis-aligned box centered at origin, 12 triangles (2 per face)

    2. `primitive_rounded_box(width: f64, height: f64, depth: f64, fillet_radius: f64, segments: u32) -> TriangleMesh`
       - Box with filleted edges and corners. Fillet radius clamped to half the smallest dimension. Generate rounded edge strips and corner patches.

    3. `primitive_cylinder(radius: f64, height: f64, segments: u32) -> TriangleMesh`
       - Centered at origin, extends from z=-height/2 to z=+height/2
       - Top and bottom cap fans + side quads (as triangle pairs)

    4. `primitive_sphere(radius: f64, segments: u32) -> TriangleMesh`
       - UV-sphere tessellation with `segments` longitude x `segments/2` latitude bands
       - Pole vertices shared (no degenerate zero-area triangles at poles)

    5. `primitive_cone(radius: f64, height: f64, segments: u32) -> TriangleMesh`
       - Apex at z=+height/2, base circle at z=-height/2
       - Bottom cap fan + side triangles from base edge to apex

    6. `primitive_torus(major_radius: f64, minor_radius: f64, major_segments: u32, minor_segments: u32) -> TriangleMesh`
       - Standard parametric torus centered at origin in XY plane

    7. `primitive_plane(width: f64, depth: f64) -> TriangleMesh`
       - Large finite rectangle in XY plane (2 triangles), at z=0
       - For analytical plane in split operations, a separate `Plane` struct with (normal, offset) will be in the split module

    8. `primitive_wedge(width: f64, height: f64, depth: f64) -> TriangleMesh`
       - Triangular prism: rectangular base, triangular cross-section (right-angle wedge)
       - 8 triangles: 2 for base, 2 for back, 2 for hypotenuse, 2 for triangle end caps

    9. `primitive_ngon_prism(sides: u32, radius: f64, height: f64) -> TriangleMesh`
       - Regular N-sided polygon extruded to height. N=6 gives hex prism. `sides` clamped to minimum 3.
       - Top/bottom cap fans + side quads

    Each function must:
    - Have comprehensive doc comments with `/// # Examples` showing construction and validation
    - Produce correct winding order (outward-facing normals via CCW convention)
    - Handle edge cases (zero dimensions produce single-point degenerate -> return minimal valid mesh or panic in debug)
    - Use `std::f64::consts::TAU` for angle calculations

    Write unit tests in the same file (`#[cfg(test)] mod tests`) verifying:
    - Each primitive is watertight: every edge shared by exactly 2 triangles
    - Triangle count matches expected values
    - Bounding box matches expected dimensions within epsilon
    - Volume is approximately correct (for simple shapes: box volume = w*h*d, sphere volume = 4/3*pi*r^3, etc.)
    - N-gon prism with sides=4 matches a box rotated 45 degrees
  </action>

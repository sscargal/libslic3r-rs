Create `csg/lattice.rs` with:

1. `pub fn cubic_lattice(cell_size: f64, strut_radius: f64, bounds: (f64, f64, f64), segments: u32) -> Result<(TriangleMesh, CsgReport), CsgError>`

   Generates a 3D grid of cylindrical struts:
   a. Compute grid dimensions: `nx = (bounds.0 / cell_size).ceil() as usize`, etc.
   b. For each grid cell, create 3 struts (one per axis): X-aligned, Y-aligned, Z-aligned cylinders.
   c. Each strut is a cylinder primitive with radius=strut_radius and length=cell_size.
   d. Position each strut at its grid location using `transform::translate`.
   e. Use `mesh_union_many` to merge all struts into a single mesh.
   f. Default segments=8 for struts (low poly for performance — lattices have many struts).

2. `pub fn gyroid_lattice(cell_size: f64, bounds: (f64, f64, f64), resolution: u32, iso_value: f64) -> Result<(TriangleMesh, CsgReport), CsgError>`

   Generates a gyroid surface mesh using marching cubes:
   a. The gyroid implicit function: `f(x,y,z) = sin(2*PI*x/cell_size)*cos(2*PI*y/cell_size) + sin(2*PI*y/cell_size)*cos(2*PI*z/cell_size) + sin(2*PI*z/cell_size)*cos(2*PI*x/cell_size) - iso_value`
   b. Create a 3D voxel grid over `bounds` with `resolution` steps per cell_size.
   c. Implement a basic marching cubes algorithm: for each voxel cube, evaluate the function at 8 corners, determine which edges are crossed (using the 256-entry lookup table), interpolate intersection points, emit triangles.
   d. The marching cubes lookup table can be hardcoded or generated. Use the classic Lorensen & Cline tables.
   e. Build TriangleMesh from all emitted triangles.
   f. `iso_value` defaults to 0.0 for the standard gyroid. Positive values thicken, negative thin the surface.

   Note: Marching cubes is standard and well-documented. The lookup table is 256 entries x 16 edge indices. Implement it cleanly with doc comments referencing the algorithm.

3. Register module in `csg/mod.rs`, re-export both functions.

4. Write tests in `crates/slicecore-mesh/tests/csg_lattice.rs`. Include a demonstration test that intersects a lattice with a solid cube to show the use case.

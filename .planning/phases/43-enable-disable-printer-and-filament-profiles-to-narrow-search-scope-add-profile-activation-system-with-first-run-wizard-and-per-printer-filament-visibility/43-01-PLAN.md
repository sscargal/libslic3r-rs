Add two methods to the `impl ProfileResolver` block in `crates/slicecore-engine/src/profile_resolve.rs`:

1. **`filter_enabled` method:**
   ```rust
   /// Filter a list of resolved profiles to only those that are enabled.
   ///
   /// Profiles are matched by type and name against the `EnabledProfiles` config.
   /// If `enabled` is `None`, returns all profiles unfiltered (for --all mode).
   pub fn filter_enabled(
       profiles: &[ResolvedProfile],
       enabled: Option<&EnabledProfiles>,
   ) -> Vec<ResolvedProfile> {
       match enabled {
           None => profiles.to_vec(),
           Some(ep) => profiles
               .iter()
               .filter(|p| ep.is_enabled(&p.profile_type, &p.name))
               .cloned()
               .collect(),
       }
   }
   ```
   Note: This is a static method (no `&self`) because it operates on already-resolved profiles. The caller provides the EnabledProfiles.

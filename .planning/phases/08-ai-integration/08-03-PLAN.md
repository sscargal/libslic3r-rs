**geometry.rs** -- Feature extraction:

Define `GeometryFeatures` struct (Serialize, Deserialize, Debug, Clone):
- `bounding_box_min: [f64; 3]` -- min corner [x, y, z]
- `bounding_box_max: [f64; 3]` -- max corner [x, y, z]
- `volume_mm3: f64`
- `surface_area_mm2: f64`
- `triangle_count: usize`
- `is_watertight: bool`
- `dimensions: Dimensions` -- struct with width_mm, depth_mm, height_mm (all f64)
- `aspect_ratio: f64` -- max_dim / min_dim
- `overhang_ratio: f64` -- fraction of surface area that is overhang (>45 degrees from vertical)
- `max_overhang_angle_deg: f64` -- steepest downward-facing angle in degrees
- `thin_wall_ratio: f64` -- heuristic: fraction of bounding box dims < 2mm (simplified for v1)
- `has_bridges: bool` -- heuristic: overhang_ratio > 0.05
- `has_small_features: bool` -- any bounding box dimension < 1mm
- `estimated_difficulty: PrintDifficulty` -- Easy/Medium/Hard enum

Define `Dimensions` struct (Serialize, Deserialize, Debug, Clone):
- `width_mm: f64`, `depth_mm: f64`, `height_mm: f64`

Define `PrintDifficulty` enum (Serialize, Deserialize, Debug, Clone, PartialEq):
- Easy, Medium, Hard
- serde rename_all = "snake_case"

Implement `pub fn extract_geometry_features(mesh: &TriangleMesh) -> GeometryFeatures`:
1. Call `slicecore_mesh::compute_stats(mesh)` to get MeshStats (volume, surface_area, aabb, is_watertight, triangle_count)
2. Compute dimensions from aabb: width = max.x - min.x, depth = max.y - min.y, height = max.z - min.z
3. Compute aspect_ratio = max_dim / min_dim (clamp min_dim to 0.001 to avoid division by zero)
4. Overhang analysis:
   - Get normals from `mesh.normals()`
   - For each triangle, compute dot product of normal with Z-up vector (0,0,1)
   - If dot < 0 (downward-facing), compute angle = acos(-dot)
   - If angle from vertical > 45 degrees (i.e., the face is more than 45 degrees from vertical toward horizontal), count as overhang
   - For overhang triangles, compute triangle area via cross product: `|edge1 x edge2| / 2`
   - Sum overhang area, track max angle
   - overhang_ratio = overhang_area / total_surface_area (use stats.surface_area)
5. Thin wall heuristic: count how many of [width, depth, height] are < 2.0mm, thin_wall_ratio = count / 3.0
6. has_bridges = overhang_ratio > 0.05
7. has_small_features = min_dim < 1.0
8. estimated_difficulty = classify_difficulty(overhang_ratio, min_dim, height):
   - Hard: overhang_ratio > 0.15 OR min_dim < 0.5 OR height > 150.0
   - Medium: overhang_ratio > 0.05 OR min_dim < 2.0 OR height > 80.0
   - Easy: otherwise

Add unit tests:
- Test with a unit cube (all faces axis-aligned, no overhangs expected on 4 vertical + 1 top face, 1 bottom face is downward but at 0 degrees from horizontal which IS overhang)
- Test dimensions computation
- Test aspect ratio for a cube (should be 1.0)
- Test that watertight cube reports is_watertight = true

Update lib.rs to add `pub mod geometry;` and `pub use geometry::{extract_geometry_features, GeometryFeatures, Dimensions, PrintDifficulty};`

Implement `pub fn extract_json(raw: &str) -> Result<serde_json::Value, AiError>`:
1. Try direct parse: `serde_json::from_str(raw.trim())`
2. Strip markdown code fences: try `raw.trim().strip_prefix("```json")` or `strip_prefix("```")`, then `strip_suffix("```")`, trim, re-parse
3. Find first `{` and last `}`, extract substring, try parse
4. If all fail: return `AiError::InvalidJsonResponse(raw.to_string())`

Implement `pub fn parse_profile_suggestion(raw: &str) -> Result<ProfileSuggestion, AiError>`:
1. Call extract_json(raw) to get serde_json::Value
2. Deserialize into ProfileSuggestion via serde_json::from_value (use default for missing fields)
3. Call validate_and_clamp() on the result
4. Return the clamped suggestion

Implement `fn validate_and_clamp(suggestion: &mut ProfileSuggestion)`:
- layer_height: clamp to 0.05..=0.3
- wall_count: clamp to 1..=6
- infill_density: clamp to 0.0..=1.0

- perimeter_speed: clamp to 20.0..=100.0

- infill_pattern: if not one of ["rectilinear", "grid", "honeycomb", "gyroid", "cubic", "monotonic"], default to "rectilinear"
- Use f64::clamp() for numeric fields

**1. Add `compute_vlh_heights` to `crates/slicecore-slicer/src/vlh/mod.rs`:**

```rust
/// Main entry point for multi-objective VLH optimization.
///
/// Pipeline:
/// 1. Sample curvature profile from mesh (reuses adaptive::sample_curvature_profile)
/// 2. Build feature map from mesh geometry
/// 3. Compute per-Z objective scores
/// 4. Run optimizer (greedy or DP based on config)
/// 5. Apply Laplacian smoothing + ratio clamping
/// 6. Optionally collect per-layer diagnostics
pub fn compute_vlh_heights(
    mesh: &TriangleMesh,
    config: &VlhConfig,
) -> VlhResult
```

Implementation:
1. Compute `mesh_max_z` from `mesh.aabb().max.z`. Return empty result for degenerate meshes.
2. Sample curvature: call `adaptive::sample_curvature_profile(mesh, config.min_height / 2.0)` (make this function `pub(crate)` in adaptive.rs).
3. Build feature map: call `features::build_feature_map(mesh, config)`.
4. Build ZSample array: for each curvature sample, compute:
   - `stress_factor` from `features::query_stress_factor(&feature_map, z)`
   - `external_surface_fraction`: compute from mesh normals at this Z (fraction of triangles whose normals face outward, i.e., `|normal.z| < 0.7` approximately -- external surfaces are steep). Simple heuristic: `1.0 - avg_abs_normal_z` at each Z. Can reuse the steepness data already computed in curvature sampling.
   - `feature_demanded_height` from `features::query_feature_demanded_height(&feature_map, z)`
   - `scores` from `objectives::compute_objective_scores(curvature, external_surface_fraction, stress_factor, config.min_height, config.max_height)`
5. Run optimizer:
   - `OptimizerMode::Greedy` -> `optimizer::optimize_greedy(&z_samples, config)`
   - `OptimizerMode::DynamicProgramming` -> `optimizer::optimize_dp(&z_samples, config)`
6. Determine pinned indices: index 0 (first layer) is always pinned. Any index where `feature_demanded_height.is_some()` is pinned.
7. Apply smoothing: `smooth::smooth_vlh_heights(&mut heights, &pinned, config.smoothing_strength, config.smoothing_iterations, 1.5, config.min_height, config.max_height)`.
8. If `config.diagnostics`, build `Vec<VlhDiagnosticLayer>` from the z_samples and final heights.
9. Return `VlhResult { heights, diagnostics }`.

pub mod objectives;

/// Normalized objective weights (always sum to 1.0).
#[derive(Debug, Clone, Copy)]
pub struct VlhWeights {
    pub quality: f64,
    pub speed: f64,
    pub strength: f64,
    pub material: f64,
}

impl VlhWeights {
    /// Create normalized weights. If all zero, defaults to quality=1.0.
    pub fn new(quality: f64, speed: f64, strength: f64, material: f64) -> Self {
        let sum = quality + speed + strength + material;
        if sum < 1e-12 {
            return Self { quality: 1.0, speed: 0.0, strength: 0.0, material: 0.0 };
        }
        Self {
            quality: quality / sum,
            speed: speed / sum,
            strength: strength / sum,
            material: material / sum,
        }
    }
}

/// Per-Z objective scores: each objective maps to a desired layer height.
#[derive(Debug, Clone, Copy)]
pub struct ObjectiveScores {
    pub quality_height: f64,
    pub speed_height: f64,
    pub strength_height: f64,
    pub material_height: f64,
}

impl ObjectiveScores {
    /// Weighted combination. Weights must be pre-normalized (sum to 1.0).
    pub fn combine(&self, weights: &VlhWeights) -> f64 {
        weights.quality * self.quality_height
            + weights.speed * self.speed_height
            + weights.strength * self.strength_height
            + weights.material * self.material_height
    }
}

/// Optimizer mode selection.
#[derive(Debug, Clone, Copy, PartialEq, Eq, Default)]
pub enum OptimizerMode {
    #[default]
    Greedy,
    DynamicProgramming,
}

Create `crates/slicecore-slicer/src/vlh/objectives.rs` with four objective functions and a combined scoring function.

**Quality objective** (`compute_quality_height`):
- Input: curvature value (0.0-1.0 range), external_surface_fraction (0.0-1.0, fraction of external surface at this Z), min_height, max_height
- Logic: `effective_curvature = curvature * external_surface_fraction` (perceptual model: internal surfaces get zero quality weight). Map effective_curvature to height: `max_height - (max_height - min_height) * effective_curvature.clamp(0.0, 1.0)`.
- This reuses the curvature-to-height mapping from `adaptive.rs` but adds the external surface weighting.

**Speed objective** (`compute_speed_height`):
- Input: min_height, max_height
- Always returns max_height (thicker = faster). This is trivial but exists for API symmetry and future extension.

**Strength objective** (`compute_strength_height`):
- Input: stress_factor (0.0-1.0, how stressed this Z region is), min_height, max_height
- Logic: stress zones get thinner layers. `max_height - (max_height - min_height) * stress_factor.clamp(0.0, 1.0)`.
- stress_factor is computed externally by the feature map (holes, thin walls, sharp angles, overhangs).

**Material objective** (`compute_material_height`):
- Input: min_height, max_height
- Always returns max_height (thicker = less material per height). Same as speed.

**Combined** (`compute_objective_scores`):
- Input: curvature, external_surface_fraction, stress_factor, min_height, max_height
- Returns `ObjectiveScores` with all four heights filled in.
- All computation is serial (no parallel reduction) for SLICE-05 determinism.

All functions are pure (no state, no randomness) ensuring deterministic output.

1. Create crates/slicecore-fileio/src/export.rs with:

   a. `ExportFormat` enum: `Stl`, `ThreeMf`, `Obj` -- with `#[derive(Debug, Clone, Copy, PartialEq, Eq)]`
      - Do NOT reuse the input MeshFormat enum (it has StlBinary/StlAscii distinction irrelevant for export)

   b. `pub fn format_from_extension(path: &Path) -> Result<ExportFormat, FileIOError>`:
      - Match on lowercased extension: "stl" -> Stl, "3mf" -> ThreeMf, "obj" -> Obj
      - Anything else -> FileIOError::UnsupportedExportFormat

   c. `fn triangle_mesh_to_model(mesh: &TriangleMesh) -> Result<Model, FileIOError>`:
      - Internal conversion function (not public)
      - Iterate mesh.vertices(), add as f32 to lib3mf_core::model::Mesh (v.x as f32, v.y as f32, v.z as f32)
      - Iterate mesh.indices(), add as Triangle
      - Create Model::default(), add Object with ResourceId(1), ObjectType::Model, Geometry::Mesh
      - Add BuildItem with object_id ResourceId(1), transform glam::Mat4::IDENTITY
      - Return Model

   d. `pub fn save_mesh_to_writer<W: Write + Seek>(mesh: &TriangleMesh, writer: W, format: ExportFormat) -> Result<(), FileIOError>`:
      - Convert mesh to Model via triangle_mesh_to_model
      - Match on format:
        - ExportFormat::ThreeMf -> model.write(&mut writer)
        - ExportFormat::Stl -> BinaryStlExporter::write(&model, writer)
        - ExportFormat::Obj -> ObjExporter::write(&model, writer)
      - Map errors to FileIOError::WriteError(e.to_string())
      - NOTE: Writer needs Write+Seek because Model::write() (3MF/ZIP) requires Seek. STL and OBJ writers only need Write but accepting Write+Seek is fine since File and Cursor both implement it.

   e. `pub fn save_mesh(mesh: &TriangleMesh, path: &Path) -> Result<(), FileIOError>`:
      - Detect format from extension via format_from_extension
      - Open file with File::create(path)
      - Delegate to save_mesh_to_writer with BufWriter wrapping the file
      - NOTE: BufWriter<File> implements both Write and Seek

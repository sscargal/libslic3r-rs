       - Table format: columns for Config, Time, Filament (g), Filament Cost, Total Cost

       - Support json/csv/markdown/table output formats
       - Use comfy-table for table format consistent with existing display code

    4. Add `pub enum OutputFormat { Table, Json, Csv, Markdown }` to analysis_display.rs or common location if not already defined. Helper function `fn determine_output_format(json: bool, csv: bool, markdown: bool) -> OutputFormat`.

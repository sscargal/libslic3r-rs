```rust
#[derive(Debug, Clone, Serialize)]
pub struct DiffEntry {
    pub key: String,
    pub display_name: String,
    pub category: Option<SettingCategory>,
    pub tier: Option<Tier>,
    pub left_value: serde_json::Value,
    pub right_value: serde_json::Value,
    pub units: Option<String>,
    pub affects: Vec<SettingKey>,
    pub description: String,
}

#[derive(Debug, Clone, Serialize)]
pub struct DiffResult {
    pub left_name: String,
    pub right_name: String,
    pub entries: Vec<DiffEntry>,
    pub total_differences: usize,
    pub category_counts: BTreeMap<String, usize>,
}
```

**Core functions:**

1. `fn flatten_value(prefix: &str, value: &serde_json::Value, out: &mut BTreeMap<String, serde_json::Value>)` -- recursively flattens a JSON Value::Object into dotted-key map. For non-object values, inserts at the current prefix. For objects, recurses with `prefix.key` (or just `key` if prefix is empty).

2. `pub fn diff_configs(left: &PrintConfig, right: &PrintConfig, left_name: &str, right_name: &str) -> DiffResult` -- serializes both configs to serde_json::Value, flattens each to BTreeMap, iterates all keys from both maps, compares values with `!=`, creates DiffEntry for each difference. After collecting raw diffs, calls `enrich_entry` on each. Computes `category_counts` from the enriched entries. Sets `total_differences = entries.len()`.

3. `fn enrich_entry(entry: &mut DiffEntry)` -- looks up `setting_registry().get_by_str(&entry.key)`. If found: populates `display_name` from `def.display_name`, `category` from `Some(def.category)`, `tier` from `Some(def.tier)`, `units` from `def.units.clone()`, `affects` from `def.affects.clone()`, `description` from `def.description.clone()`. If not found: `display_name` stays as raw key, category/tier/units remain None/empty.

4. `pub fn format_value(value: &serde_json::Value, units: &Option<String>) -> String` -- formats a JSON value for human display:
   - `Value::Number(n)` -> format as number, append ` {units}` if units is Some
   - `Value::String(s)` -> just the string (no quotes)
   - `Value::Bool(b)` -> "true"/"false"
   - `Value::Array(arr)` -> comma-separated formatted values (no brackets)
   - `Value::Null` -> "(none)"
   - `Value::Object(_)` -> serde_json::to_string (fallback)

**toolpath.rs** — Add `Support` variant to `FeatureType` enum. Update all match arms in toolpath.rs.

**traditional.rs** — Implement traditional support structure generation:

1. `project_support_regions(overhang_regions_per_layer: &[Vec<ValidPolygon>], layers: &[SliceLayer], build_plate_only: bool) -> Vec<Vec<ValidPolygon>>`:
   - For each layer with overhangs, project the overhang region downward through all layers below it to the build plate (or to the first model surface if build_plate_only=false).
   - At each layer below, the support cross-section is the overhang region. Intersect with the complement of the model contours at that layer to avoid support inside the model.
   - Use `polygon_difference(support_region, &model_contours_at_layer)` to clip support away from model interior.
   - Return per-layer support regions (union of all projected overhangs reaching each layer).

2. `apply_xy_gap(support_regions: &[ValidPolygon], model_contours: &[ValidPolygon], xy_gap_mm: f64) -> Vec<ValidPolygon>`:
   - Inward-offset support regions by xy_gap to create clearance from model walls.
   - Use `offset_polygons(support_regions, -mm_to_coord(xy_gap_mm), JoinType::Miter)`.
   - Additionally, expand model contours by xy_gap and subtract from support: `polygon_difference(offset_support, &expanded_model)`.
   - Return gap-adjusted support regions. Filter out any regions that collapse to empty.

3. `generate_support_infill(support_regions: &[ValidPolygon], density: f64, pattern: SupportPattern, layer_index: usize, extrusion_width: f64) -> Vec<InfillLine>`:
   - Generate infill lines within support regions using the configured pattern.
   - For `SupportPattern::Line`: single-direction parallel lines (no alternating angle — lines in one direction for easy peeling). Use 0-degree angle.
   - For `SupportPattern::Grid`: cross-hatched rectilinear using existing `generate_infill(InfillPattern::Grid, ...)` dispatch.
   - For `SupportPattern::Rectilinear`: alternating 0/90 rectilinear using existing `generate_infill(InfillPattern::Rectilinear, ...)` dispatch.
   - Support density is typically 10-20% (sparse). Compute spacing from density and extrusion_width.
   - Return infill lines for this layer's support region.

4. `generate_traditional_supports(overhang_regions: &[Vec<ValidPolygon>], layers: &[SliceLayer], config: &SupportConfig, extrusion_width: f64) -> Vec<Vec<SupportRegion>>`:
   - Main entry point for traditional support generation.
   - Project overhang regions downward.
   - Apply XY gap at each layer.
   - Generate support infill per layer.
   - Package results as `Vec<Vec<SupportRegion>>` (per-layer).

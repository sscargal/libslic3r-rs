    Create `crates/slicecore-engine/src/builtin_profiles.rs`:

    1. Use `include_str!` pattern with inline TOML const strings (not external files -- keeps binary self-contained).

    2. Built-in profiles (Claude's discretion on exact values, use sensible defaults):
       - `generic_pla`: filament, nozzle_temp=210, bed_temp=60, standard PLA settings
       - `generic_petg`: filament, nozzle_temp=240, bed_temp=80, PETG settings
       - `generic_abs`: filament, nozzle_temp=250, bed_temp=100, ABS settings
       - `generic_printer`: machine, 220mm bed, 0.4mm nozzle, 250mm/s max, Marlin dialect
       - `standard`: process, 0.20mm layer height, 20% infill, moderate speeds (per user decision)

    3. `BuiltinProfile` struct: name, profile_type, toml_content (static &str).
    4. `get_builtin_profile(name: &str) -> Option<&'static BuiltinProfile>` for exact lookup.
    5. `list_builtin_profiles() -> &'static [BuiltinProfile]` for enumeration.

    Create `crates/slicecore-engine/src/config_validate.rs`:

    1. `ValidationSeverity` enum: Warning, Error.
    2. `ValidationIssue` struct: field (String), message (String), severity (ValidationSeverity), value (String).
    3. `validate_config(config: &PrintConfig) -> Vec<ValidationIssue>`:
       - Warning: layer_height > nozzle_diameter
       - Warning: layer_height > nozzle_diameter * 0.75 (very thick layers)
       - Warning: speeds > 500 mm/s (any speed field)
       - Warning: infill_density > 100 or < 0
       - Error: nozzle temp > machine max (if machine defines max)
       - Error: nozzle temp > 350 (absolute safety limit)
       - Error: bed temp > 150 (absolute safety limit)
       - Error: layer_height <= 0
       - Error: nozzle_diameter <= 0

    4. `resolve_template_variables(gcode: &str, config: &PrintConfig) -> String`:
       - Replace {nozzle_temp}, {bed_temp}, {first_layer_nozzle_temp}, {first_layer_bed_temp}, {layer_height}, {nozzle_diameter} with values from config.
       - Unknown {variables} left unchanged.

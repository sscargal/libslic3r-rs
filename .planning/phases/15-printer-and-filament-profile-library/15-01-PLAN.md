   a. **Types:**
   ```rust
   #[derive(Debug, Clone, serde::Serialize, serde::Deserialize)]
   pub struct ProfileIndexEntry {
       pub id: String,           // e.g., "orcaslicer/BBL/filament/Bambu_PLA_Basic_BBL_A1"
       pub name: String,         // Original profile name
       pub source: String,       // "orcaslicer" or "bambustudio"
       pub vendor: String,       // e.g., "BBL", "Creality"
       pub profile_type: String, // "filament", "process", "machine"
       pub material: Option<String>,      // Extracted from name for filament profiles
       pub nozzle_size: Option<f64>,      // Extracted from name
       pub printer_model: Option<String>, // Extracted from @-suffix
       pub path: String,                  // Relative path to TOML file
       pub layer_height: Option<f64>,     // Extracted from process name
       pub quality: Option<String>,       // "fine", "standard", "draft", etc.
   }

   #[derive(Debug, Clone, serde::Serialize, serde::Deserialize)]
   pub struct ProfileIndex {
       pub version: u32,  // Always 1
       pub generated: String, // ISO 8601 timestamp
       pub profiles: Vec<ProfileIndexEntry>,
   }

   #[derive(Debug)]
   pub struct BatchConvertResult {
       pub converted: usize,
       pub skipped: usize,
       pub errors: Vec<String>,
       pub index: ProfileIndex,
   }
   ```

   b. **Inheritance resolution function:**
   ```rust
   /// Resolve the inheritance chain for a profile within its vendor/type directory.
   /// Loads the profile and all its ancestors, merging from root to leaf.
   fn resolve_inheritance(
       profile_path: &Path,
       type_dir: &Path,  // e.g., .../BBL/filament/
       cache: &mut HashMap<String, ImportResult>,
   ) -> Result<ImportResult, EngineError>
   ```
   - Read and parse the JSON profile at `profile_path`
   - Check `"inherits"` field -- if present, find the parent JSON file in `type_dir` (filename = `{inherits_value}.json`)
   - Recursively resolve parent, caching results in `cache` HashMap
   - Use `merge_import_results(&[parent_result, child_result])` to overlay child on parent
   - If parent file not found, start from `ImportResult` with default PrintConfig
   - Handle up to 10 levels of inheritance depth (guard against circular references)

   c. **Batch conversion function:**
   ```rust
   pub fn batch_convert_profiles(
       source_dir: &Path,     // e.g., /home/steve/slicer-analysis/OrcaSlicer/resources/profiles
       output_dir: &Path,     // e.g., profiles/orcaslicer
       source_name: &str,     // "orcaslicer" or "bambustudio"
   ) -> Result<BatchConvertResult, EngineError>
   ```
   - Walk `source_dir` finding vendor directories (direct children that are directories, skip .json files, skip non-directory entries like .py scripts and .json metadata)
   - For each vendor, walk `filament/`, `process/`, `machine/` subdirectories
   - For each `.json` file, parse JSON and check `"instantiation"` field -- skip if not `"true"`
   - Call `resolve_inheritance()` to get a fully resolved ImportResult
   - Call `convert_to_toml()` to get TOML output
   - Sanitize filename: replace spaces with `_`, remove `@`, remove parentheses, replace `/` with `_`
   - Write TOML to `output_dir/vendor/type/sanitized_name.toml`, creating directories as needed
   - Build `ProfileIndexEntry` for each converted profile using helper extractors
   - On individual profile conversion errors, log the error string and continue (don't abort batch)
   - Return `BatchConvertResult` with counts and the complete `ProfileIndex`

   d. **Metadata extraction helpers** (all `pub(crate)`):
   - `sanitize_filename(name: &str) -> String` -- replace problematic chars
   - `extract_material_from_name(name: &str) -> Option<String>` -- match against known material list: PLA-CF, PLA+, PLA, PETG-CF, PETG, ABS, ASA, PA-CF, PA, PC, TPU, PVA, HIPS, PP, etc. Match longest first to avoid "PLA" matching before "PLA-CF"
   - `extract_layer_height_from_name(name: &str) -> Option<f64>` -- regex-like match for `0.XXmm` pattern at start of name
   - `extract_nozzle_size_from_name(name: &str) -> Option<f64>` -- match `X.X nozzle` pattern
   - `extract_printer_model(name: &str) -> Option<String>` -- extract text after `@` prefix (e.g., "@BBL X1C" -> "BBL X1C")
   - `extract_quality_from_name(name: &str) -> Option<String>` -- match "Extra Fine", "Fine", "High Quality", "Standard", "Draft", "Super Draft" (case-insensitive)

   e. **Index generation function:**
   ```rust
   pub fn write_index(index: &ProfileIndex, output_dir: &Path) -> Result<(), EngineError>
   ```
   - Serialize `ProfileIndex` to pretty JSON
   - Write to `output_dir/index.json`

   f. **Index loading function:**
   ```rust
   pub fn load_index(profiles_dir: &Path) -> Result<ProfileIndex, EngineError>
   ```
   - Read and parse `profiles_dir/index.json`
